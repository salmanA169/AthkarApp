# AthkarApp
- Knowing prayer times and notify 
- read athkar from Quran and sunnah
- counter althker
 # Libraries
 
 - Room for the data persistence.
 - Work Manager for background job.
 - Kotlin Coroutines.
 - Kotlin Flows.
 - Gson for serialization/deserialization.
 - Dagger Hilt dependency injection.
 - MVVM architecture. 
 - Retrofit
